# DNA Analyzer

## Задача заказчика
Приложение для студентов-биологов для анализа ДНК последовательностей.

## Стек
Python 3.11, Tkinter, pytest

## Запуск

```bash
pip install -r requirements.txt
python src/main.py
```

## Тесты

```bash
pytest
```

## Структура
- src/core — логика
- src/ui — интерфейс
- tests — автотесты
- data — sample.fasta
- docs — документация

## Что умеет
- Проверка ДНК
- GC%
- Подсчёт нуклеотидов
- Обратный комплемент
- Транскрипция в РНК
- Сохранение отчёта

## Ограничения MVP
Нет графиков состава и поддержки очень больших файлов.
